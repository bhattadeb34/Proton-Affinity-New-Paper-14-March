# Proton Affinity Prediction: Multi-Fidelity Machine Learning

**Authors**: Debjyoti Bhattacharya, Wesley F. Reinhart  
**Affiliation**: Materials Science and Engineering, Pennsylvania State University

## Overview

Multi-fidelity machine learning for proton affinity (PA) prediction, combining:

- **High-fidelity**: B3LYP/def2-TZVP DFT with full thermochemistry  
- **Low-fidelity**: PM7 semi-empirical calculations  
- **Ground truth**: NIST experimental PA (1185 molecules) and site-specific DFT PA (821 sites)

### Pipeline

1. **Featurization** → MACCS, Morgan, RDKit, PM7, optional Mordred/3D and site features  
2. **Delta learning** → predict correction Δ = target_PA − PM7_PA; final PA = PM7_PA + predicted Δ  
3. **Validation** → 5-fold CV with per-fold feature selection (variance → correlation → Lasso rank → parsimonious k)

## Data

The `data/` directory is **not** included in the repo (~6.9 GB). Expected layout:

- **1185_molecules** (~37 MB): NIST dataset; PM7 outputs; B3LYP results; `Dataset.csv` with SMILES and experimental PA.  
- **251_molecules** (~6.9 GB): Site-level PM7 and B3LYP outputs.

Place your data in `data/1185_molecules/` and `data/251_molecules/` as required by the loaders in `code/data_loading/`.

## Run order

1. **Featurization** (needs `data/`):
   ```bash
   python run_featurization.py
   ```
   Writes `processed/featurization/features_1185.parquet`, `features_251.parquet`, and metadata.

2. **Delta-learning cross-validation** (optional; long run):
   ```bash
   python run_delta_cv.py
   ```
   Runs 4 variants (NIST-i/ii, Site-i/ii), 9 models, 5-fold CV. Writes `processed/model_outputs/delta_cv_*.json` and `delta_cv_summary.csv`.

3. **NIST validation** (optimal k, learning curve, plots):
   ```bash
   python run_nist_validation.py
   ```
   Uses existing CSVs if present; otherwise runs analysis then plotting. Writes `processed/analysis/nist_features_vs_perf.csv`, `nist_learning_curve.csv`, `nist_validation_plots.pdf`. If CV results exist, the plot includes median selected features.

## Methods

- **Delta learning**: \( \text{PA}_{\text{final}} = \text{PA}_{\text{PM7}} + \hat{\Delta} \), with \( \Delta = \text{PA}_{\text{target}} - \text{PA}_{\text{PM7}} \).  
- **Feature selection** (per fold): variance filter → correlation filter → Lasso rank → parsimonious k (1-SE rule).  
- **Models**: RidgeCV, LassoCV, ElasticNetCV, BayesianRidge, DecisionTree, RandomForest, GradientBoosting, XGBoost, MLPRegressor.
